### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（学生）、维修员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 知识信息
5. 公告信息
6. 留言板
7. 后台管理
8. 个人中心
9. 我的收藏

**维修员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 维修订单管理
6. 维修进度管理
7. 评价信息管理



**管理员功能：**
1. 登录
2. 系统首页（宿舍人数统计、每日报修人数统计、满意度统计）
3. 个人中心（修改密码、个人信息）
4. 学生管理
5. 维修员管理
6. 楼栋信息管理
7. 宿舍信息管理
8. 宿舍分配管理
9. 报修信息管理
10. 维修订单管理
11. 维修进度管理
12. 评价信息管理
13. 知识分类管理
14. 知识信息管理
15. 留言板管理
16. 系统管理（关于我们、系统简介、轮播图管理、公告信息）
